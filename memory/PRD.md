# Start Up 13 - Website PRD

## Original Problem Statement
Build a professional bilingual (English/Arabic) website for Start Up 13 software development company (startup13.net). Website inspired by Instagram visual identity (@start_up13).

## User Personas
- **Business Owners**: Looking for software development services
- **Startups**: Needing digital solutions and mobile apps
- **Enterprise Clients**: Seeking custom software solutions

## Core Requirements
- Bilingual support (English/Arabic) with RTL
- Professional dark theme with cyan accent
- Single page layout with smooth scroll
- Contact form saving to MongoDB
- Responsive design for all devices

## Tech Stack
- Frontend: React + Tailwind CSS + Shadcn/UI
- Backend: FastAPI + Motor (async MongoDB)
- Database: MongoDB
- Fonts: Outfit, Tajawal, Inter
